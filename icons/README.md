# HireBot Icons

This directory contains the extension icons in different sizes:

- icon16.png - 16x16 pixels (toolbar)
- icon32.png - 32x32 pixels (extension management page)  
- icon48.png - 48x48 pixels (extension management page)
- icon128.png - 128x128 pixels (Chrome Web Store)

## Icon Design

The HireBot icon features:
- A professional briefcase or person silhouette
- Blue and purple gradient colors (#667eea to #764ba2)
- Clean, modern design
- Good contrast for visibility
